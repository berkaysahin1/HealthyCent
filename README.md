## Kullanım

Öncelikle aşağıdaki komutu terminale yazarak gereken kütüphaneleri yüklüyoruz.
```` console
$ flutter pub get
```` 
Sonrasında ise 
```` console
$ flutter run
```` 
komutu ile uygulamamızı çalıştırabiliriz.

Bunlar dışında ek bir yükleme veya ayar yapılması gerekmemektedir.
